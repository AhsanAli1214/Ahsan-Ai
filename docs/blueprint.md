# **App Name**: AI Hub Express

## Core Features:

- Home Screen: Displays a grid or list of available AI tools and resources.
- Tool Detail Screen: Shows detailed information about a selected AI tool, including description, usage instructions, and relevant links.
- Category Navigation: Allows users to browse AI tools by category.
- Search Functionality: Enables users to search for specific AI tools or resources.
- Resource Recommendations: AI-powered tool that recommends relevant AI tools based on user activity and interests, such as allowing users to provide some information to get curated recommendation of specific tools. This system relies on a vector database containing the embeddings and textual descriptions of various AI tools.
- User Account Management: Allows users to create and manage their accounts, save favorite tools, and personalize their experience.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to create a calm and intellectual atmosphere.
- Background color: Light gray (#F5F5F5) for a clean, modern look.
- Accent color: Muted teal (#80CBC4) for highlighting important elements and calls to action.
- Headline font: 'Poppins', sans-serif, for headings and titles to maintain a modern and readable aesthetic.
- Body font: 'Inter', sans-serif, for all paragraphs of text, long running or short.
- Use a consistent set of minimalist icons to represent different tool categories and actions.
- Maintain a clean, grid-based layout with sufficient white space for readability and ease of navigation.
